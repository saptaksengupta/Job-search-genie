# Job-search-genie
A simple Job Search Demo Application By using ReactJs as frontend and NodeJs as backend.

# Backend Intro
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Api Endpoint Documentation (Swagger-UI Link):
```bash
http://localhost:9998/api
```
